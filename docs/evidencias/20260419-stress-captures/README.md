# Evidencias complementares de stress test, Jenkins e JMeter

Esta pasta consolida evidencias geradas em 19/04/2026 para complementar a entrega da disciplina Integracao Continua e DevOps.

## Arquivos principais

- `grafana-during-stress-dashboard.png`: print principal do Grafana durante o teste de carga.
- `grafana-during-stress-screen.png`: copia do print principal do Grafana durante o teste de carga.
- `jenkins-login.png`: Jenkins instanciado e acessivel em `http://localhost:8088`.
- `jenkins-dashboard.png`: segunda captura do Jenkins local.
- `jenkins-compose-status.txt`: status do container Jenkins no Docker Compose.
- `jenkins-http-check.txt`: resposta HTTP da tela de login do Jenkins.
- `jmeter-dashboard.png`: relatorio HTML do Apache JMeter atualizado apos execucao contra a API atual.
- `k6-stress-test.log`: nota sobre a execucao do stress test com k6.

## Observacao
