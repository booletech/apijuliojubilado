Pipeline versionado no Jenkinsfile

Checkout Obtém o código-fonte versionado no repositório.
Testes dos Backends Executa testes Maven das APIs em paralelo.
Build do Front-end Instala dependências e gera build de produção.
Publicar Imagens Publica as imagens Docker no Docker Hub.
Preparar Runtime Gera parâmetros e imagens para o Kubernetes.
Deploy no Kubernetes Aplica os manifests e coleta evidências do cluster.
Smoke Test Valida endpoint da aplicação após o deploy.
Stress Test Execução opcional do k6 após a entrega.
Arquivo de referência: Jenkinsfile, na raiz do repositório.
